# SmartExplorerInstall 3.0.0

Installer package for the SmartCash 3.0.0 Explorer.

## Contents

- `smart-iquidus-install.sh`
- `iquidus-explorer.service`
- `nginx.conf`
- `README.md`
- `SMART_IQUIDUS_EXPLORER_COMPLETE_GUIDE.md`

## Purpose

- Installs the SmartCash Core binaries.
- Sets up MongoDB, Nginx and the Iquidus Explorer service.
- Uses the community Explorer configuration and public seed nodes.

## Use

```bash
sudo bash smart-iquidus-install.sh
```
